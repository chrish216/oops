hexo-config

Posted on 16-03-06 In hexo Reading time ≈ 1 mins.

GOAL: config Hexo-Next theme.

Site config

Find ./_config.yml .

# Extensions
theme: next

# Site
title: chrish-dev
author: chrish

# URL
permlink: :category/:title

# Writing
highlight:
  enable: false

# Date / Time Format
date_format: YY-MM-DD

# Post meta
post_meta:
  updated_at:
    enable: false

Theme config

Find ./themes/next/_config.yml .

# Menu Settings
menu:
  home: / || home
  tags: /tags/ || tags

# Scheme Settings
scheme: Mist

# Font Settings
global:
  family: PT Mono
  size: 0.65
codes:
  family: Menlo
  size: 1

# Sidebar Settings
sidebar:
  position: left

# Sidebar Avatar
avatar:
  url: /images/avatar.jpg

# Other Settings
codeblock:
  highlight_theme: solarized

Create tags and categories pages.

$ hexo new page tags
$ cat << EOF > source/tags/index.md
---
title: all-tags
type: tags
---

EOF
$ hexo new page categories
$ cat << EOF > source/tags/index.md
---
title: all-categories
type: categories
---

EOF

Custom CSS

CSS are defined in ./themes/next/source/css/ .

Add custom style support in main.styl .

// My Layer
// --------------------------------------------------
@import "_my/mycss";

Create a new file _my/mycss.styl for custom CSS.

p {
    margin: 0 0 10px 0;
}

.posts-expand .post-title {
    font-size: 1.35em;
}

h2 {
    font-size: 1.25em;
}

h3 {
    font-size: 1.15em;
}

.post-body {
    word-break: break-all;
}
